16x16 Matrix - 
Turning indicators/strobe light for a 16x16 LED Matrix (2, 8x16 Adafruit LED Backpacks). Includes a Processing visualizer for designing custom blinky patterns.

16x24 Matrix (original, 2012) - 
Original Arduino sketch for the bike circuit containing LED turn indicators, brake light, speedometer, and odometer. Also includes a Processing LED visualizer for designing custom light patterns.
